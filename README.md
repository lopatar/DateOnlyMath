# DateOnlyMath

This package adds extension methods for the [DateOnly struct](https://learn.microsoft.com/en-us/dotnet/api/system.dateonly?view=net-7.0).

# Extension methods
- Add
- Subtract
- Multiply
- Divide
- BiggerThan
- LowerThan

# Installation
[NuGet](https://www.nuget.org/packages/DateOnlyMath/)
